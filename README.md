# Brew-Haven-Caf-

## Cafe Sales Dashboard Analysis (Excel)

This project focuses on analyzing sales performance for a cafe using Microsoft Excel.  
The goal was to extract meaningful business insights and evaluate profitability and return on investment (ROI) through a simple yet effective dashboard.

### Key Insights
- Analysis of total revenue, total cost, and net profit
- Calculation of ROI to measure business performance
- Sales trends over time (daily / monthly)
- Identifying best-performing periods and low-performing days
- Tracking historical performance using date-based analysis

### Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Data Visualization
- Basic financial calculations (Profit, ROI)
